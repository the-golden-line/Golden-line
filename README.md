
# THE GOLDEN LINE – Punkte-Tracker

## Nutzung über GitHub Pages (iOS-sicher)

1. Neues Repository auf GitHub erstellen (z. B. `golden-line`)
2. Diese ZIP entpacken
3. `index.html` in das Repository hochladen
4. In GitHub:
   Settings → Pages → Branch: main / root
5. Öffne die erzeugte URL in Safari
6. Safari → Teilen → Zum Home-Bildschirm

Danach läuft die App wie eine native App.
